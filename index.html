ℹ About TUSK.AI
TUSK.AI is a Qualtrics Survey Builder that reads questionnaire documents and generates valid QSF files with full logic support.

Auto-updates: This app is hosted online. Refresh the page to get the latest version — no reinstall needed.

Supported formats: PDF, DOCX, DOC, XLSX, XLS, CSV, TXT
Features: Display logic, Skip logic, Scoring, Validation, Matrix questions, NPS, Rank Order
